# Chorus Module

This directory contains the documentation and research for the TillySynth Chorus engine, a meticulously modeled Bucket Brigade Device (BBD) effect inspired by the Roland Juno-60.

## 📖 Article: Understanding the Juno-60 Chorus

I have restructured the research from the original notebook into a clean, web-ready article. It covers historical context, the mathematics of the BBD effect, and the specific implementation details for the Juno I, II, and I+II modes.

### View the Article
To open the article in your browser, run:
```bash
open article/index.html
```

## 🧪 Research & DSP Prototyping

- **[chorus.ipynb](chorus.ipynb)**: The original Jupyter notebook containing Python prototypes for the Delay Buffer, LFO, and BBD filtering logic.
